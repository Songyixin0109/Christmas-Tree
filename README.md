# 🎄 Three.js 圣诞树项目

这是一个基于 Three.js 制作的精美3D圣诞树效果，包含粒子系统、发光效果和交互控制。

## 📁 项目结构

```
Christmas Tree/
├── model/                    # 3D模型文件夹
├── shader/                   # 着色器文件
├── three/                    # Three.js 相关类
├── three.js-master/          # Three.js 库文件
├── Christmas Tree.html       # ✅ 可直接运行版本
└── index.html               # 🔧 开发版本
```

## 🚀 快速开始

### 方式一：直接运行（推荐给普通用户）
- 双击打开 `Christmas Tree.html` 文件
- 无需任何额外配置，直接在浏览器中运行
- 适合分享给不熟悉代码的朋友

### 方式二：开发模式
- 使用 **Live Server** (VSCode插件) 或本地服务器打开 `index.html`
- 需要服务器环境，因为涉及本地文件加载
- 适合开发者进行修改和调试

## ✨ 特色功能

- 🎄 可自定义的粒子圣诞树
- ⭐ 树顶发光星星
- ✨ 粒子发光和闪烁效果
- 🌠 流星雨特效
- 🎛️ 实时参数调节面板
- 🔄 平滑的旋转动画
- 💫 后期处理发光效果

## 🎮 交互控制

打开右侧控制面板可以调节：
- 圣诞树参数（粒子数量、大小、高度等）
- 星星参数（颜色）
- 后期处理效果（发光强度、曝光度等）
